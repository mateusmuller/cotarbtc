# cotarBtc
This script will fetch the Bitcoin information from Mercado Bitcoin API.

## How to use it?
Use:

* *$ ./cotarBtc.sh 10 10*
  
It will execute 10 times each 10 seconds.

## Installation?
* $ git clone https://github.com/mateusmuller/cotarbtc
* $ cd cotarbtc
* $ ./cotarbtc.sh 1 5
